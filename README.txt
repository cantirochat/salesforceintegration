# Cantiro Chatbot Backend

This is a Flask-based backend that integrates an OpenAI assistant with Salesforce. The backend handles incoming user data and creates new leads in Salesforce.

## Features
- Accepts POST requests with user details and conversation summaries.
- Sends data to Salesforce using their REST API.

2. Install dependencies:
